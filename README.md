# 个人Hyprland配置文件
## 所需软件包
- `hyprland` 混成器 
- `hyprlock` 锁屏程序
- `hypridle` 闲置守护程序
- `hyprpaper` 壁纸程序
- `waybar` 状态栏
- `foot` 终端
- `tofi` 菜单
- `mako` 通知
## 安装并使用
**ArchLinux**
```
pacman -S hyprland hyprpaper waybar foot tofi mako hyprlock hypridle
```
将配置文件复制到`$HOME/.config/`内即可

hyprpaper指向`~/Picture/Wallpaper/main.jpg`
hypridle指向 `~/Picture/me/cutie.jpeg`
